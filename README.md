# vue3 项目模板

# 启动项目

```
npm install
npm run dev

会启动前端服务(2333端口)和后端服务(9999端口)。
```

# 项目依赖

- [x] @element-plus
- [x] axios
- [x] pinia
- [x] vue-router
- [x] sass
- [x] typescript
